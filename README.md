\# Summary

The project demonstrates the use of a rubber ducky payload to connect a Meterpreter reverse shell using PowerShell. The payload uses Metasploit-Framework to initiate the reverse shell.



Some components may require manual configuration and prior knowledge of common security frameworks.



\# Requirements

* Basic Metasploit-Framework knowledge.
* Basic familiarity with scripting.



\# Overview

* Manual changes in the payload must be made for the payload to be functional.
* The payload uses a Metasploit-framework module (Web-Delivery Module) to operate on the victims system. 
* Operates on windows 10 and 11.
* Admin Privileges are required for the use of the payload.
